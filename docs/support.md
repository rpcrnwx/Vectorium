## Issue Reporting

### Bug Reports

1. **Required Information**
   ```markdown
   ## Bug Report Template
   
   ### Description
   Clear description of the issue
   
   ### Steps to Reproduce
   1. Step one
   2. Step two
   3. Step three
   
   ### Expected Behavior
   What should happen
   
   ### Actual Behavior
   What actually happens
   
   ### Environment
   - OS: [e.g., Windows 10]
   - Browser: [e.g., Chrome 120]
   - Version: [e.g., 1.0.0]
   ```

2. **Debug Information**
   ```typescript
   // Enable debug logging
   const debug = {
     level: 'debug',
     enabled: true,
     console: true,
     file: true
   };
   ```

### Feature Requests

1. **Request Format**
   ```markdown
   ## Feature Request Template
   
   ### Problem Statement
   What problem does this solve?
   
   ### Proposed Solution
   How should it work?
   
   ### Alternatives Considered
   What other approaches exist?
   
   ### Additional Context
   Any other relevant information
   ```

2. **Priority Levels**
   - Critical: Business blocking
   - High: Major feature
   - Medium: Enhancement
   - Low: Nice to have
